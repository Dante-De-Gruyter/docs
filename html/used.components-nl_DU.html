Used Components

Name Comment
Postfix Mail Transfer Agent (MTA)
Dovecot POP3, IMAP en Managesieve server
Nginx, Nginx Web server
OpenLDAP, ldapd(8) LDAP server, gebruikt voor het opslaan van e-mail accounts (optioneel)
MySQL, MariaDB, PostgreSQL SQL server gebruikt voor het opslaan van applicatiedata. Kan ook gebruikt worden om e-mail accounts op te slaan.
mlmmj Mailing lijst beheerder. Wordt met iRedMail-0.9.8 en latere versies gebundeld.
Amavisd-new Interface tussen Postfix en SpamAssassin, ClamAV.
SpamAssassin Content-gebaseerde spam scanner
ClamAV Virus scanner
Roundcube webmail Webmail (PHP)
SOGo Groupware Een groupware die kalender (CalDAV), contactenlijst (CardDAV), taken en ActiveSync services voorziet.
Fail2ban Scant logbestanden en verbant IPs die kwaadwillende tekens vertonen.
iRedAPD Een simpele postfix policy server ontworpen door het iRedMail team, met SRS (Sender Rewrite Scheme) ondersteuning.
